# User Management API

## 🚀 Project Startup

### 🛠 Requirements
- Node.js
- Firebase project with Firestore enabled
- serviceAccountKey.json created (see below)

### 📦 Installation
```bash
npm install
```

### ⚙️ Run the Application
```bash
npm run dev
```

### 🔑 Firebase Service Account
Place `serviceAccountKey.json` in the root directory.
